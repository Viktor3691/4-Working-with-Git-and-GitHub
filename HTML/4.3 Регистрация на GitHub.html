4.3 Регистрация на GitHub

В этом видео уроке я покажу вам как зарегистрировать аккаунт на сайте GitHub и мы создадим наш первый онлайн репозиторий. После этого мы загрузим наш проект в этот онлайн репозиторий и будем в дальнейшем с ним работать через терминал в редакторе VS Code и графический интерфейс редактора.

Регистрация на GitHub

Зарегистрироваться на GitHub можно на официальном сайте платформы — github.com. Для регистрации нужен почтовый адрес в зоне .com, например почта Google. При попытке зарегистрировать аккаунт на почту в зоне .ru появляется ошибка Unable to verify your captcha response — «Не удалось проверить ваш ответ капчи».

Пошаговая инструкция

Чтобы создать учётную запись на GitHub, нужно:

  1. На главной странице найти и нажать кнопку Sign up (обычно находится в правом верхнем углу) — это приведёт на страницу регистрации.
  2. Ввести адрес электронной почты — он будет использоваться для проверки аккаунта и важных уведомлений.
  3. Создать пароль — GitHub рекомендует использовать комбинацию букв в верхнем и нижнем регистре, цифр и специальных символов для повышения безопасности.
  4. Пройти проверку на то, что пользователь не робот (CAPTCHA) — это предотвратит автоматические регистрации.
  5. Нажать кнопку «Создать учётную запись» — GitHub отправит код проверки на указанный адрес электронной почты. Нужно проверить inbox (и папку со спамом, если необходимо), а затем ввести код на сайте GitHub, чтобы подтвердить почту.
  6. По желанию — GitHub может задать несколько вопросов о том, как пользователь планирует использовать платформу (для обучения, личных проектов или профессиональной работы). Можно ответить на эти вопросы, чтобы получить рекомендации, или пропустить этот шаг.
  7. Выбрать тарифный план — GitHub предлагает бесплатный план, который включает неограниченное количество общедоступных и частных репозиториев. Платные планы добавляют расширенные функции и увеличенное хранилище, но они необязательны и могут быть добавлены позже.
